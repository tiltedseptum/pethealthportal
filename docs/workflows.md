# Core Workflows

## 1. Log observation
- Add observation with notes, date and optional image
- Save entry
- Create timeline event

## 2. Add vet visit
- Add visit summary
- Attach prescription/report
- Save entry
- Create timeline event

## 3. Review timeline
- View chronological events
- Filter by type/date
- Understand health progression

## 4. Check preventive care
- View upcoming and overdue items
- Mark item as completed

## 5. Retrieve medical records
- Search or filter records
- Open file quickly
